Aula 3 - Testes Estatísticos Simples no R

O que aprendemos?

1. Teste t para Amostras Independentes

  • Importar dados diretamente do Google Sheets
  • Reorganizar dados de formato longo para wide (pivot_wider)
  • Realizar Teste t (comparação entre dois grupos independentes)
  • Visualizar os resultados com boxplots e p-values

2. Teste t para Amostras Dependentes (Pareado)

  • Reorganizar dados para teste pareado
  • Aplicar o Teste t pareado
  • Avaliar normalidade e variâncias com Shapiro-Wilk e Teste F

3. ANOVA (Comparação de Múltiplos Grupos)

  • Ajustar um modelo ANOVA com função aov()
  • Avaliar resíduos: histograma, Shapiro-Wilk, Bartlett e Levene
  • Comparação de médias usando emmeans e cld()

4. Alternativas Não Paramétricas

  • Teste de Kruskal-Wallis
  • Uso do pacote agricolae para múltiplas comparações após Kruskal

5. Diagnóstico de Resíduos

  • Uso do pacote DHARMa para avaliar resíduos de modelos
